# Running processes:

### Frontend:
```
cd frontend
npm run dev
```

### Backend (server):
```
cd backend
ts-node server.ts
```

# CLI Commands:

### Linting:
```
npm run lint:check
npm run lint:fix
```

### Prettier formatting:
```
npm run prettier:check
npm run prettier:fix
```

### To run TypeScript files:
```
ts-node {file}.ts
```
